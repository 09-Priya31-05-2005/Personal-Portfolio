Projects

Network Access Registration system

NARS (Network Access Registration System) is a web-based platform designed to streamline the process of registering student-owned devices for campus LAN access. Students can log in using their email credentials and submit essential details like MAC and IP addresses. Developed using HTML, CSS (front-end), and PHP with phpMyAdmin (back-end), NARS replaces manual processes with a secure, automated system. It simplifies device registration, enhances connectivity management, and ensures reliable network access. The goal is to improve the campus experience by promoting a more connected and digitally integrated environment.

Automatic Evaluation of MathBERT Embeddings of Student's Algebra Question

Automates grading of descriptive algebra answers using MathBERT embeddings and machine learning models. We transformed student responses into 385-dimensional embeddings and tested classifiers like SVM, KNN, Decision Trees, and Random Forest. SVM achieved the best performance across accuracy, precision, and F1-score. The system enhances grading consistency, reduces manual effort, and provides quick feedback. Feature selection improved model reliability, making this approach suitable for scalable academic evaluation.

Vein Detector-Automated Blood Drawing System

This project involves designing a portable, low-cost vein detection system to assist in automated blood collection and injection. Using infrared imaging with a Raspberry Pi 4B and NoIR Camera, the device identifies veins not visible to the naked eye. The prototype processes real-time images to highlight veins using machine learning and image processing. Simulations were done on TinkerCAD, and the final prototype uses 940nm IR LEDs and photodiodes. Future enhancements include projecting vein maps on the skin and improving battery efficiency. The system aims to improve accuracy, hygiene, and patient comfort in clinical and rural healthcare environments.

Dual Hormone Artificial Pancreas

Dual Hormone Artificial Pancreas-Bi Hormonal pump that delivers Both Insulin and Glucagon. The project introduces a dual hormone artificial pancreas designed to regulate blood glucose by delivering both insulin and glucagon, mimicking the natural pancreatic function. Unlike conventional single-hormone systems, this bi-hormonal pump offers better control over both high and low glucose levels, preventing hypoglycemia effectively. Using an ESP32 microcontroller, a TCS3472 color sensor, and an OLED display, the system detects glucose concentrations in real time via colorimetric reactions. Polynomial regression was used to calibrate sensor readings with near-perfect accuracy. Based on the glucose level, relay modules activate the appropriate hormone pump‚Äîinsulin for high and glucagon for low levels. The setup includes medical-grade components such as silicone tubing and peristaltic pumps for hormone delivery. The prototype successfully demonstrates real-time monitoring, decision-making, and response, replicating an artificial pancreas function. This system showcases a cost-effective, responsive, and user-friendly medical simulation. The project confirms the feasibility of using embedded systems and sensor-based automation in biomedical device design.
